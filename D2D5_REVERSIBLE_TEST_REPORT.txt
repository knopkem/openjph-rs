ROOT CAUSE ANALYSIS

The test failures indicate a BUG IN THE RUST IMPLEMENTATION:

1. SCOPE: Affects reversible (lossless) 5/3 DWT encoding/decoding
2. PATTERN: Only tests with NO decomposition pass perfectly
3. MSE RANGE: 10,900-2.1B depending on bit depth and decomposition level
4. IMPLICATION: Data is being corrupted during DWT processing

Potential areas for investigation:
- 5/3 filter implementation (forward or inverse transform)
- DWT level computation or recursion logic
- Coefficient quantization for reversible mode
- Memory access patterns (alignment, overflow)
- Decomposition level iteration
- Coefficient rounding in reversible mode
